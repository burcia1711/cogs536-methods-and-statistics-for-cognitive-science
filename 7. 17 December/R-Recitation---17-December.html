1 1 Foundations of ANOVA

1.1 1. Quick data inspection checklist

  • Students often skip checking factor types and end up with numeric IVs.
  • In ANOVA, predictors should be factors, outcome should be numeric.
# A minimal "inspection" template
inspect_data <- function(df) {
  cat("---- STRUCTURE ----\n")
  print(str(df))
  cat("\n---- SUMMARY ----\n")
  print(summary(df))
  cat("\n---- MISSING VALUES ----\n")
  print(colSums(is.na(df)))
}

1.2 2. Between-subject vs within-subject (quick intuition)

  • Between-subject: each participant contributes ONE score (per Dependent Variable) for the condition they are in.
  • Within-subject: each participant contributes MULTIPLE scores (repeated measures).

1.3 3. One-way between-subject ANOVA (core)

1.3.1 Example: Viagra dose Libido

This is a small dataset similar to the lecture example: three groups (Placebo/Low/High) and a numeric DV.

viagra <- data.frame(
  Dose = factor(rep(c("Placebo", "Low", "High"), each = 10),
                levels = c("Placebo", "Low", "High")),
  Libido = c(
    # Placebo
    3,2,1,1,4,2,3,1,2,4,
    # Low
    5,2,4,2,3,4,5,3,4,2,
    # High
    7,4,5,3,6,5,6,4,7,5
  )
)

1.4 4. Assumptions (and the residuals, not raw data note)

  • Normality check is typically applied to residuals (or model errors), not each raw group.
  • Homogeneity of variance is commonly checked via Levenes test.

1.4.1 4.1 Normality of residuals

shapiro.test(residuals(m_oneway))
## 
##  Shapiro-Wilk normality test
## 
## data:  residuals(m_oneway)
## W = 0.92651, p-value = 0.0397
qqnorm(residuals(m_oneway)); qqline(residuals(m_oneway))

1.4.2 4.2 Homogeneity of variance (Levene)

# If not installed, run: install.packages("car")
library(car)
leveneTest(Libido ~ Dose, data = viagra)
## Levene's Test for Homogeneity of Variance (center = median)
##       Df F value Pr(>F)
## group  2  0.0672 0.9352
##       27

1.5 5. Planned contrasts (orthogonal comparisons)

  • Contrasts are hypothesis-driven comparisons.
  • For 3 groups, we can test:
    1. Placebo vs (Low + High)
    2. Low vs High

1.5.1 Approach 1: Set custom contrasts in R

viagra_c <- viagra
# Contrast 1: Placebo vs (Low+High)  -> (-2, +1, +1)
# Contrast 2: Low vs High           -> (0, -1, +1)
contrasts(viagra_c$Dose) <- cbind(
  Placebo_vs_Treated = c(-2, 1, 1),
  Low_vs_High        = c(0, -1, 1)
)

m_contrast <- aov(Libido ~ Dose, data = viagra_c)
summary(m_contrast)
##             Df Sum Sq Mean Sq F value   Pr(>F)    
## Dose         2  42.87  21.433   14.43 5.46e-05 ***
## Residuals   27  40.10   1.485                     
## ---
## Signif. codes:  0 '***' 0.001 '**' 0.01 '*' 0.05 '.' 0.1 ' ' 1

1.5.2 Approach 2 (optional): Directly test specific comparisons using emmeans

If you want a more flexible workflow for follow-ups (still based on the aov fit).

# install.packages("emmeans")
library(emmeans)

emm <- emmeans(m_oneway, ~ Dose)
emm
##  Dose    emmean    SE df lower.CL upper.CL

1.6 6. Post-hoc tests + multiple comparisons corrections

1.7 7. One-way repeated-measures ANOVA (core)

1.7.1 Example B: Repeated measures across 3 conditions

Here we create a within-subject dataset in long format: - Subject: participant ID (factor) - Condition: within-subject factor - Score: DV

set.seed(1)
rm_data <- expand.grid(
  Subject   = factor(1:18),
  Condition = factor(c("A", "B", "C"), levels = c("A", "B", "C"))
)

# Simulate a pattern: B > A, C highest
rm_data$Score <- with(rm_data,
  rnorm(nrow(rm_data),
        mean = ifelse(Condition == "A", 50,
               ifelse(Condition == "B", 55, 60)),
        sd = 6)
)

head(rm_data)
##   Subject Condition    Score
## 1       1         A 46.24128
## 2       2         A 51.10186
## 3       3         A 44.98623
## 4       4         A 59.57168
## 5       5         A 51.97705
## 6       6         A 45.07719

m_rm <- aov(Score ~ Condition + Error(Subject/Condition), data = rm_data)

1.8 8. When assumptions fail: Friedmans ANOVA (non-parametric RM)

  • Friedman is an alternative when repeated-measures assumptions are badly violated.
  • Friedman requires wide-ish input (or you can reshape). Well reshape longwide here.
rm_wide <- reshape(rm_data, timevar = "Condition", idvar = "Subject", direction = "wide")
head(rm_wide)
##   Subject  Score.A  Score.B  Score.C
